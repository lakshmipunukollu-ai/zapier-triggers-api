# Story 7: React Dashboard (Optional)

## As an operator, I want a visual dashboard to monitor event flow and troubleshoot delivery issues.

### Acceptance Criteria
- React SPA with event explorer
- View events with filters (status, source, event_type)
- View event detail with delivery log
- View system metrics (counts, charts)
- View subscriptions
- Connect to backend API

### Technical Notes
- Minimal but functional dashboard
- Uses fetch/axios to call backend API
- Basic styling (can use Tailwind or simple CSS)
